# Melhorias Futuras

Esta pasta concentra ideias e especificações para evoluções do sistema que **ainda não foram implementadas**. Cada arquivo descreve um conjunto coerente de funcionalidades com escopo, justificativa e esboço de design — para serem retomadas quando fizer sentido.

Não é código. Não é documentação oficial do produto. É um roadmap vivo, escrito em português, do que vem por aí.

## Índice

### Módulo Prestadores de Serviço

A Fase 1 (MVP — cadastro + avaliação digital) já está em produção. Os arquivos abaixo descrevem extensões planejadas:

- **[prestadores-fase-2-documentos-e-alertas.md](./prestadores-fase-2-documentos-e-alertas.md)** — Documentos vinculados (PES, ASO, NR-18) com alertas de validade, taxa de conformidade por prestador, widget de não-conformidades no Dashboard.

- **[prestadores-fase-3-integracoes.md](./prestadores-fase-3-integracoes.md)** — Integração com o bot Telegram (avaliação por foto), vínculo de comprovantes de pagamento, relatório PDF consolidado para auditoria PBQP-H, bloqueio automático após não-conformidades repetidas.

## Como usar

1. Quando uma ideia daqui for priorizada, mover a especificação para uma issue/PR e iniciar a implementação.
2. Depois de implementada, remover o arquivo desta pasta (ou marcar como "✅ Implementada em vX.Y").
3. Novas ideias podem ser adicionadas a qualquer momento — só lembrar de atualizar o índice.
